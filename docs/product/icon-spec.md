# Icon Spec

## Requirements
- Readable at 16px and 32px
- Terminal + guidance motif (command prompt with review/check overlay)
- Works on dark and light backgrounds
- Recognizable silhouette at small sizes

## Export Set
- `32x32.png` — taskbar, favicon
- `128x128.png` — app list
- `128x128@2x.png` — Retina/HiDPI
- `icon.ico` — Windows
- `icon.icns` — macOS

## Notes
- Placeholder icons acceptable for dev builds
- Final icons should be created before v1.0 release
